Arquitectura de Despliegue

El proyecto consta de tres componentes principales desplegados en la nube:

  • Base de datos: MongoDB Atlas (cluster gratuito M0 en región Sao Paulo)
  • Backend API: Render.com (servicio gratuito con Node.js)
  • Frontend: Netlify (SPA con Vue 3)

Enlaces de Acceso

Frontend (Netlify): https://taupe-tanuki-3cd1bb.netlify.app

Backend API (Render): https://mercapp-api-e0a7.onrender.com

Guía de Ejecución Local

Requisitos:

  • Node.js (versión 18 o superior)
  • npm
  • MongoDB Atlas o local

Pasos:

  1. Clonar repositorio: git clone https://github.com/Menestra1700/Mercapp.git
  2. Instalar backend: cd backend && npm install
  3. Configurar variables de entorno en archivo .env
  4. Ejecutar backend: node server.js
  5. Instalar frontend: cd frontend && npm install
  6. Ejecutar frontend: npm run dev
  7. Abrir navegador en http://localhost:5173

Nota sobre el Despliegue

El backend fue desplegado en Render en lugar de Railway debido a errores persistentes con el builder de Railway (npm not found). La funcionalidad es equivalente y la API está completamente operativa.
